# 척척밥상

상품정보를 제공 받아 사용자에게 페이지를 보여주는 프론트엔드 과제 프로젝트입니다.

---

## 기술 스택

- **Next.js**
- **TypeScript**


---

## 프로젝트 구조

```txt
src/
 ├ app/                    # Next.js App Router
 │  ├ layout.tsx           # 루트 레이아웃
 │  ├ page.tsx             # 홈 페이지
 │  ├ globals.css          # 전역 CSS
 │  └ favicon.ico          # 파비콘
 ├ assets/                 # 정적 자산
 │  ├ fonts/               # 폰트 파일
 │  ├ icons/               # SVG 아이콘
 │  └ images/              # 이미지 파일
 ├ components/             # 컴포넌트
 │  └ common/              # 공통 컴포넌트
 └ const/                  # 관련 상수 정의

```
